# License policy

No repository-wide license has been selected yet.

Until that decision is made:

- Do not copy third-party implementation code into this repository.
- Keep external projects as references or patch targets only.
- Record the origin, commit and license of any future copied/generated component.
- Inkscape-derived modifications and patches must remain compatible with Inkscape's license.
- GTK modifications must preserve the applicable LGPL notices.
- Projects with AGPL or custom/source-available licenses may be used for architectural study, but their code must not be copied without an explicit compatibility review.
